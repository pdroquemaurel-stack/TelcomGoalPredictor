# AGENTS.md

## Product intent
This project is a football match prediction app for a pan-African audience.
The product should feel mobile-first, game-oriented, sponsor-friendly, and easy to demo.

## Engineering priorities
- Preserve Render deployability
- Keep Prisma changes minimal
- Prefer incremental changes over rewrites
- Prioritize UX clarity over feature sprawl
- Make saved prediction state obvious and reliable

## UX priorities
- Stronger homepage
- Better mobile player experience
- Clear prediction states
- Visible gamification
- Simple but credible admin dashboard

## Working style
- Audit first
- Plan second
- Implement third
- Verify main flows before finishing
